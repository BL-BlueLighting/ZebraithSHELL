## 这是什么？
Zebraith SHELL 是一个基于 Python 的 Terminal Emulator (终端模拟器) 程序。

## 怎么用？
Zebraith SHELL 与普通 TE 不同的是，普通 TE 依靠其他由不同编程语言写成的其他程序来支撑，而 Zebraith SHELL 用 Plugin 系统开放了插件生态。

### 说人话
说人话就是 Zebraith SHELL 有插件系统，不是程序驱动。

#### 还是听不懂
呵呵哒。请看下面的列表：

| 原 | 比 |
| ---| --- |
| Zebraith SHELL | 泡面桶 |
| 基本插件 | 面 |
| 操作系统 | 水 |
| 其他插件 | 调料包 |

这些东西组合起来，才泡好了一桶面（Zebraith SHELL の 完全体）

#### 听懂了，所以到底怎么用
额额额，很简单。
写插件，安装插件，用插件。
自带的插件都比较简陋，不适合日常使用。
所以推荐自己写一写插件。
